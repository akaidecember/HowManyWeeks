# How Many Weeks

A static single-page site that maps a date of birth and year span into a grid of weeks.

## Run Locally

Open `index.html` in a browser. No build step, package install, or backend is required.

## Free Hosting

This project is ready for static hosting:

- GitHub Pages: push these files to a repository and enable Pages from the repository settings.
- Netlify: drag the project folder into Netlify Drop or connect the repository.
- Vercel: import the repository as a static project.
- Cloudflare Pages: connect the repository and leave the build command empty.

The site uses only HTML, CSS, and JavaScript, so it can be hosted on any free static host.
